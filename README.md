# RFactor-2026-Swerve-Code

Current Working code of Team RFactor for the 2026 Season.

### Key program features include:
- Field relative and heading controlled swerve drive using YAGSL
- Swerve and Vision Odometry
- Pathplanner autonomous
- Motor control using PID and feedforward
- Automatic setpoints for subsystems
- Automatic aiming and shooting using vision odometry and a lookup table
